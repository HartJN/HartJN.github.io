Grid VS Flexbox

Flexbox vs Grid

If you have recently tried your hand at styling a website, you've probably ran into either of these two. Both of these properties handle how elements are layed out on a webpage, but both have their own strengths and weaknesses, making it important to choose the correct one.

What is flexbox?

Flexbox was designed for creating one-dimensional layouts, best suiting components that are arranged in a single row or column.

What is Grid?

Grid was designed as a way to created two-dimensional layouts, best suited for complex layouts. A good way to explain this, is to compare it to a newspaper, you have multiple rows and columns of content layed out on the pages.

Which should you use?

Both grid and flex have their own uses! It depends on the situation, do you need a simple, one-dimensional layout like a menu or form? if so, flexbox is the way to go! Do you need to create a complex layout, that has multiple rows and columns? then grid is the better choice for you.

Learning CSS Grid and Flexbox

Here are some games that teach you CSS grid and Flexbox in an interactive way!

Grid Garden

Grid Garden teaches you about the different parts of the CSS grid layout in 28 different interactive levels.

Grid Attack

Grid Attack is a game designed to teach you all aspects of CSS Grid. It includes 80 levels, 40 real-project layouts and 3 game modes (easy, medium, hard) all in a fun and interactive environment!

Flexbox Froggy

Flexbox Froggy teaches you how to learn the flexbox syntax. Over 24 levels, it shows you the wide range of properties that flexbox offers, and how to use them.

Flex Box Adventure

Flex Box Adventure is from the same makers of Grid Attack, it is an interactive adventure game, consisting of 24 levels made to assist you in understanding the basics of Flex Box!
